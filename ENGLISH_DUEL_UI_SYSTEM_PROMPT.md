You are a senior UI/UX director with expertise in:
• Competitive gaming interfaces
• EdTech psychology
• FinTech minimal design systems
• High-performance mobile UI systems

You are designing the visual identity for a mobile app called:

ENGLISH DUEL

This is NOT a casual language app.
This is a competitive AI-powered 1v1 voice battle platform.

The UI must communicate:

• Competition
• Skill mastery
• Confidence
• Modernity
• Speed
• Clean precision
• Subtle prestige

It must NOT feel:

• Childish
• Cartoonish
• Academic
• Over-gamified
• Heavy
• Colorful chaos

---

🎯 VISUAL PHILOSOPHY

Design Direction:

“Minimal Competitive Intelligence”

This is a blend of:
• Chess.com seriousness
• Esports lobby confidence
• FinTech clarity
• Apple-level cleanliness

The app should feel like:
A serious skill arena.

---

🎨 COLOR SYSTEM (psychology-based)

Background:
#F6F7FB (very soft cool neutral)
Creates cognitive calm and reduces visual fatigue.

Surface:
Pure white (#FFFFFF)
Cards float with subtle elevation.

Primary Accent:
Soft electric violet range (#6E6AF6 family)
Represents intelligence, modern AI, uniqueness.

Use purple sparingly.
Never flood the screen with it.

Secondary Accent:
Very light neutral greys (#F1F2F7 range)
For inactive states.

Success:
Subtle emerald (not bright green)
Competitive win, not childish reward.

Danger:
Muted red, never aggressive neon.

Avoid:
• Bright blue
• Orange
• Neon green
• Gradients
• Heavy shadows

---

🔹 ELEVATION & DEPTH

Use extremely subtle shadows.

Shadow style:
Blur 24–30
Low opacity
Offset downward slightly

Cards must feel floating, not heavy.

No thick borders.
Use light separators only if necessary.

---

🔹 TYPOGRAPHY SYSTEM

Font style:
Modern geometric sans (Inter / SF style).

Hierarchy:

Headline:
Bold, confident, clean.
Used for:
• Username
• Score
• Result title

Body:
Medium weight.
Readable.
Slightly tightened letter spacing.

Micro text:
Light grey tone.

Avoid:
• Decorative fonts
• Rounded playful fonts
• Condensed fonts

---

🔹 RADIUS SYSTEM

Small elements:
12–16 radius

Cards:
20–24 radius

Pills / chips:
Fully rounded (28+)

Buttons:
Soft but structured.

The app must feel “rounded but controlled”
Not bubbly.

---

🔹 MICRO-INTERACTIONS

All transitions:
150–200ms

Use:
• Subtle scale animation (1.0 → 1.02)
• Soft fade transitions
• Smooth elevation change

Avoid:
• Bounce
• Elastic curves
• Over-animation

Battle screen will later be more intense,
but MVP remains calm.

---

🔹 LAYOUT RULES

Use generous whitespace.
Breathing space is premium.

Avoid:
• Tight padding
• Dense layout
• Visual noise

Follow 8pt grid system.

Spacing scale:
8 / 12 / 16 / 20 / 24

---

🔹 COMPETITIVE SIGNALING

Visual cues for competition:

• Opponent vs You layout symmetry
• Clear timer positioning
• Score displayed confidently
• Rank badges minimal but strong

No cartoon trophies.
No emoji overload.

---

🔹 EMOTIONAL JOURNEY

Home:
Calm, confident.

Searching:
Anticipation, subtle pulse.

Live duel:
Focused, minimal distraction.

Scoring:
Tension.

Result:
Clear outcome, controlled emotion.

Winning should feel earned.
Not flashy.

---

🔹 ACCESSIBILITY

Contrast must be strong enough for readability.
Avoid light grey on white for primary content.
Text must remain readable under sunlight.
