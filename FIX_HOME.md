Redesign the Home screen of an AI-powered English speaking duel app.

Goal:
Make it feel competitive, dynamic, and social-first, while still keeping a premium AI education vibe.

Current structure:
- Greeting (Hello, Munkh)
- Stat pills (#128, 62%, streak, B2)
- "Start a Duel" card with Quick Duel button
- Silver tier progress
- Recent matches list
- Bottom navigation

Problems:
- Too static
- Too card-heavy
- Not emotionally engaging
- Lacks competitive energy
- Feels like a dashboard instead of an arena

New Direction:
Competitive Social Dominant (60%)
Premium AI Secondary (40%)

Design Instructions:

1. HERO SECTION (Arena Feel)
- Remove heavy grey card background.
- Make the hero section borderless and immersive.
- Add a subtle gradient or radial glow behind the CTA.
- Headline: Strong, competitive tone (e.g., "Ready to dominate?")
- Primary CTA must feel powerful (larger, more contrast, subtle glow).
- Add a dynamic element like:
  - 🔥 3 win streak
  - ⚡ Rank +5 this week
  - 🎯 2 challenges waiting

2. STAT HIERARCHY
- Make Rank visually dominant.
- Reduce emphasis on B2 level (make it minimal).
- Win rate and streak should look energetic.
- Use clear hierarchy: Big number → small label.

3. SOCIAL ENERGY STRIP
Add a horizontal strip below hero:
- "2 new challenges"
- "Top 10 this week"
- "Daily quest 80%"
- "You moved up 3 places"

This strip should feel alive and scrollable.

4. TIER PROGRESSION
- Keep progress bar but modernize it.
- Make it look like season progression, not enterprise dashboard.
- Add countdown tension: "18 days left".

5. RECENT MATCHES
- Remove heavy cards.
- Make it clean list style.
- Add subtle win/loss indicator.
- Add rank movement indicator if possible.

6. VISUAL STYLE
- Light background (#F7F8FC or similar).
- Primary deep blue.
- Add one energetic accent (cyan or violet).
- Reduce borders.
- Increase whitespace.
- Make it feel like a competitive arena, not a productivity tool.

7. EMOTIONAL GOAL
When opening Home:
- User should feel challenged.
- User should feel progress.
- User should feel movement.

Avoid:
- Overuse of gaming neon
- Cheap gradients
- Overly playful visuals
- Too many cards

Output:
Provide a new Home layout structure and visual description.
Do not write code.
Focus only on UI/UX redesign.
